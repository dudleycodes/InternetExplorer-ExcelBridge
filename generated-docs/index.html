ExcelBridge.js

Provide a real-time link between Internet Explorer and Microsoft Excel via ActiveX enabling basic document manipulation and data retrieval using JavaScript. Purposely written to have no external dependencies.

Example

var ExcelLink = new excelBridgeFromFile("\\server1\docs\abc.xlsx");
ExcelLink.setSelected('Sheet2');
var data = ExcelLink.getRow(1, 0, 'Sheet3');
ExcelLink.setCell('B4', 'Data has been read successfully.', 'Sheet1');
ExcelLink.terminate();
